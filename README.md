# 开饭啦(在线点餐系统)

## 项目介绍

本项目开发语言采用 Java 1.8, 后端框架使用 SSM, 前端框架使用 Layui, 数据库使用 MySQL 8.0 和 Redis。
该项目分为管理端和用户端,管理端分为用户管理、分类管理、菜品管理、订单管理等功能;用户端分为菜品浏览、加入购物车、清空购物车、下单、订单管理、个人中心(
修改个人信息、修改密码)。
